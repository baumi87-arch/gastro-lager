Verbindung einrichten

Verbinde diese App mit deiner Airtable-Datenbank. Du brauchst nur 2 Minuten.

Gehe zu airtable.com/create/tokens → „Create token" → Scope: data.records:read + data.records:write → deine Base auswählen
In der Airtable URL: airtable.com/appXXXX/...

So findest du die Daten

1

Gehe zu airtable.com/create/tokens und erstelle einen neuen Token mit Read+Write Berechtigung auf deine Base

2

Die Base ID findest du in der URL wenn du deine Base öffnest: appXXXXXXXX

3

Tabellenname = genau so wie deine Tabelle in Airtable heißt (Standard: Lagerbestand)
